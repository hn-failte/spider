# Spider

## 使用

下载源码：
`git clone git@github.com:hn-failte/spider.git` 或 [直接下载](https://github.com/hn-failte/spider/archive/master.zip)

安装依赖：
`npm install`

使用：
`node [爬虫名]`

封ip恢复下载：
等待一段时间后直接再次执行`node [爬虫名]`，会自动恢复到上次下载的位置
